# Scribble-supervised Learning for Medical Image Segmentation

This repository provides re-implementations of several scribble-supervised methods for medical image segmentation:
 
1. [DMPLS – MICCAI 2022](https://arxiv.org/abs/2203.02106) ✅
2. [ShapePU – MICCAI 2022](https://arxiv.org/pdf/2206.02118) ✅
3. [UAMT - Pattern Recognition 2022](https://www.sciencedirect.com/science/article/pii/S0031320321005215) ✅
4. [ScribbleVC - ACM MM 2023](https://drive.google.com/file/d/1IqvUslNZAMa6-wvCL67clh7W981iXh4t/view)
5. [ScribFormer - TMI 2024](https://arxiv.org/pdf/2402.02029) ✅
6. [DMSPS - MedIA 2024](https://www.sciencedirect.com/science/article/abs/pii/S1361841524001993?dgcid=author) ✅
7. [ScribbleVS - ArXix 2024](https://arxiv.org/pdf/2411.10237) ✅
8. [TABNet - ArXiv 2025](https://arxiv.org/pdf/2507.02399)
